# TDD Kata
- Init bundle
  ```bash
  bundle init
  ```
- Add "gem 'rspec'" in Gemfile
- Install bunble
  ```bash
  bundle install
  ```
- Exec rspec
  ```bash
  bundle exec rspec --init
  ```
- Run all test
  ```bash
  bundle exec rspect
  ```
- Run a specific test
  ```
  bundle exec <ful_path_file_.rb>
  ```
